    Spese in dettaglio

    Se una spesa non è divisibile esattamente (es. dividendo per 3), gli importi vengono arrotondati per difetto al centesimo e i centesimi restanti assegnati a chi, al momento del calcolo, sta spendendo di meno: è normale che due persone con la stessa spesa "sulla carta" abbiano un centesimo di differenza. Se il totale anticipato per una spesa/cena non coincide con il dovuto calcolato, la differenza viene ridistribuita automaticamente (solata se si è pagato di più, controsolata se si è pagato di meno).

    ⚙️ Token GitHub

    Istruzioni amministratore:
    Accedere su GitHub → Settings → Developer settings → Personal access tokens (fine-grained, con permesso "Contents: Read and write" sul repo). Il token resta salvato solo in questo browser (non su GitHub Pages, non visibile ad altri visitatori) e rimane impostato anche se chiudi la pagina o il browser, finché non lo rimuovi. Oppure più velocemente, cliccando al seguente link GeneraToken, si viene reindirizzati al sito per creare il token.

    Istruzioni utenti:
    Inserire nello spazio sottostante il token fornito dall'amministratore. Una volta fatto, chiudere il menù a tendina e non modificare più il campo.
