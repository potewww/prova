Amministrazione

Le funzioni qui sotto richiedono il token amministratore: permette di creare nuovi siti per i rimborsi, rinominarli o eliminarli.

⚙️ Token amministratore GitHub
Consigliato: un fine-grained token con Resource owner = la tua organization, Repository access = "All repositories", permessi Administration (org) + Contents + Pages in read/write. Così il token non può toccare i tuoi repository personali, solo quelli dell'organization. Vedi il README per i dettagli (e per l'alternativa più semplice ma meno sicura col token classico). Resta salvato solo in questo browser. Oppure più velocemente, cliccando al seguente link GeneraToken, si viene reindirizzati al sito per creare il token.
⚠️ js/config.js non è ancora stato configurato con owner/repo/org: finché resta così, la creazione di nuovi eventi non funzionerà. Vedi il README.
